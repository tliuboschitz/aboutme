Todo sobre las etiquetas de HTML   

NIEDERST ROBBINS, Jennifer. HTML5 Pocket Reference Quick, Comprehensive, Indispensable (Pocket Reference (O'Reilly)). 5ta. Ed. Págs. 1 a 4, 6 y 7. EE.UU.: O'Reilly Media, 2013. ISBN: 978-1449363352  

Orientaciones para la lectura:

  • ¿Qué significa HTML y cuál es su función principal en la creación de páginas web?  

  • ¿Cuál es la última especificación de HTML y cuáles son las principales características que introduce?  

  • ¿Cuál es la diferencia entre las etiquetas <head> y <body> en un documento HTML?  

  • ¿Cómo se utiliza la etiqueta <header> y en qué se diferencia de otras etiquetas de encabezado?  

  • ¿Qué elemento HTML se usa para representar el contenido principal de una página web y cómo se usa?  

  • ¿Cuál es el propósito de las etiquetas <div> y <span> en HTML? ¿En qué se diferencian?  

  • ¿Cuál es la utilidad de las etiquetas <section> y <article> en la estructura de un documento HTML?  

  • ¿Cuál es la importancia de las etiquetas <h1>, <h2>, <h3>, etc., en HTML para la jerarquía de contenido? 

  • ¿Cómo se usa la etiqueta <p> para representar párrafos de texto en un documento HTML? 

  • ¿Cómo se crea un hipervínculo en HTML utilizando la etiqueta <a>? 

  • ¿Qué atributos se utilizan con la etiqueta <img> para agregar imágenes a una página web? 

  • ¿Cómo se estructura una tabla en HTML utilizando las etiquetas <table>, <tr>, <th>, y <td>? 

  • ¿Cuál es la diferencia entre las etiquetas <ol> (lista ordenada) y <ul> (lista desordenada) en HTML? 

  • ¿Qué etiquetas se utilizan para crear un formulario en HTML y cuál es su propósito? 

  • ¿Cómo se crea un campo de entrada de texto en un formulario utilizando la etiqueta <input>?
